توضيح الخطة التدريبية






تعليمات الجدول :

  • الرمز ( س . أ ) : ساعات الاتصال الأسبوعي "ساعات المحاضرات ".
  • الرمز ( مح ) : عدد ساعات المحاضرات النظرية.
  • الرمز ( عم ) : عدد ساعات المحاضرات العملي والورش.
  • الرمز ( تم ) : عدد ساعات المحاضرات التمارين .
  • الرمز ( و . م ) : الوحدات المعتمدة " ساعات المادة ".
  • متطلب باللون الأحمر : متطلب ذو أهمية .


نصائح وإرشادات :

  • عند تسجيل المواد احرص على إضافة المواد التي تكون متطلب لمادة أخرى بالفصل القادم.
  • المواد المهمة "باللون الأحمر " مواد تخصصية ولها متطلبات احرص على إضافتها .
  • عند التزامك بالخطة التدريبية لكل فصل ستنهي برنامجك التعليمي بالوقت المحدد.
